Analysis for San Francisco Bay Ford GoBike Service 201902¶

by Akubazgi Gebremariam¶

Investigation Overview¶

I wanted to look at the characteristics or features of of Ford GoBike datast that could be used to improve bike service usage and identify the main users in the given time series datasets. To do this, the advanced python libraries are applied. Accordingly, insights are identified.

Dataset Overview¶

This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. The dataset contains arround 13 columns and latter some columns are added to make it more suitable for investigation. Some records associated with null values are removed because they were small in number. Handling datatype issue has benn alos made.

Bike usage analysis interms of user type¶

The dataset contains 174952 of records in the 201902 given data. Almoss, most of the user type are Subscribers. The following shows the number of user type in percentage.

  • Customer : 90.5%
  • Subcriber : 9.5%

Thus most of the user types are permanent users who agrees to receive emails or notifications to use the bike ford company servies.

# User type analysis using pie plot
def pie_plot(column:pd.DataFrame, title):
        plt.figure(figsize=(10,6))
        sorted_counts = column.value_counts()
        plt.pie(sorted_counts, labels = sorted_counts.index, startangle = 90,
                counterclock = False, autopct='%1.2f%%');
        plt.axis('square')
        #plt.legend('user_type')
        plt.title(title, fontsize=15, fontweight ='bold');

Analysis on relationship among each day and duartion in relation to user type¶

Some interesting observation here

  • Sunday is the day which users take more duration in the bike usage. This could be, people may move far from home to spend time at the break day.
